VR+mocap prototype

Combining VR and full-body motion capture for highly physical virtual experiences.

This is a prototype VR multiplayer sword dueling game. Two players are intended to face off with physical foam swords on a motion capture stage. The capture data is streamed to the VR headsets to drive visuals that align with the real world.

This prototype version only supports one VR user at the moment. I am using an OptiTrack system streaming data to a modified version of the official OptiTrack Unity plugin for 1:1 skeletal tracking. There is a lot of work to be done to improve space calibration, body tracking, latency, and the foam swords themselves (too heavy!).

My friend Hazel helped a ton with testing and foam sword creation. I used several CC0 assets from PolyHaven.com and AmbientCG.com to create a generic lava planet. Thank you!
