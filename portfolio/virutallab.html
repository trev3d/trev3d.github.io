A simulated lab space for Purdue University's Nuclear Engineering department.

This is a virtual lab project led by Purdue University Prof. Stylianos Chatzidakis. The lab is built on Unity, and I'm responsible for nearly all visuals and programming. It runs in a browser and increases accessibility to curriculum-necessary lab equipment.

Users don't interface with 2D abstractions but instead press buttons, flip switches, and turn knobs on accurate 3D representations. After using this virtual lab, a student should be able to easily use the real thing.

The program runs in a browser and is optimized for low-spec devices such as Chromebooks and tablets. There are max 25k triangles rendered in 22 draw events. The program's load size is shy of 50mb.

I collaborated with Prof. Chatzidakis and his team to program fully-functional simulations of the equipment. The program is a flexible 'sandbox' that will deal with any arrangements the user may configure. Inputs & outputs are fully simulated and everything interacts realistically.

While I was responsible for most programming and artwork, I still used some third party assets. The cable dynamics come from Obi Rope, and the radioactive source container model comes from PolyHaven.

The lab is built in Unity and I created most of the visuals using in Blender, Substance Painter, Marmoset Toolbag, and Affinity Designer.
